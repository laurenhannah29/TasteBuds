What we are MADE of:
Login/Signup

The login page authenticates a user by entering their credentials. TasteBuds only requires a username and password. If there is a new user, they can signup and create an account. For security purposes, we have added email confirmation links.

Create a Post

Users can publish any number of posts. Any user can see your post or comment as it is a public space. You can add an image by simply uploading the image you want. You can also insert a caption and select which cuisine the recipe will fall under.

Save a Post

Do not lose that tasty-looking recipe; save it for later. Saved posts do not expire but they might disappear if the original post gets deleted.

Edit Profile

Users can change their personal information and add any additional details in their bio section. This allows users to create a self-identity by adding an overview of general information and their skill sets.

Filter the Content

This tag links all posts that have the same tag, and it is clickable. When an user clicks it,they will be taken to the selected tag page with all posts with the same tag.
